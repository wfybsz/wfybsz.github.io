快速打开指定bv或av号的b站视频

发表于 2020-02-23 更新于 2021-05-28 阅读次数：

最近跟朋友互相分享视频总是用 av 号，打开浏览器再输入号码特别麻烦，于是花了点时间搞了个快速打开视频的方法。

首先建一个空的文件夹，把位置复制下来。

回桌面选中 此电脑 按下 Alt + Enter 打开属性界面。

点击左边的 高级系统设置，出现的小窗口点右下角的 环境变量，又出现的小窗口中下面那一栏 系统变量 里找到 Path 双击，又一次弹出窗口。Win7 系统直接在那一长条文本的开头粘贴刚才空文件夹的位置，然后加一个 ; （注意是英文分号）与后面的文本分隔开；Win10 系统点击右侧新建然后粘贴，Enter。所有窗口点 确定 来关掉它，然后把刚开始打开的系统属性也关掉。

这里我介绍两种版本。

需要输入 av 号

按下 Windows + R，在出现的小窗口里填上 av 79903305，Enter，浏览器自动启动并打开了 Lex 的视频。
